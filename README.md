# ☕ ChaiSpot - Link https://chai-shop-pwga.vercel.app/

ChaiSpot is a full-stack community-driven chai shop discovery and rewards platform. It allows users to discover local chai shops, add new shops, share ratings and reviews, earn reward points for contributions, and redeem those points for coupons.

## 🚀 Features

- 🔐 User Registration and Login
- 🪪 Token-Based Authentication
- 🏪 Browse Chai Shops
- ➕ Add New Chai Shops
- 📍 Store Shop Location using Latitude and Longitude
- ⭐ Rate Chai Shops
- 💬 Write and Edit Reviews
- 🎯 Earn Reward Points
- 🎁 Redeem Points for Coupons
- 🎟️ View Claimed Coupons
- 💾 Persistent Login 
- 🔄 Backend Profile Synchronization
- 📱 Responsive User Interface
